Endpoints					    Permissões

CRUD Cliente
- GET    http://locacao/api/v1/clientes         ->  Administrador
- GET    http://locacao/api/v1/clientes/id	->  Administrador
- POST   http://locacao/api/v1/clientes		->  Cliente
- PUT    http://locacao/api/v1/clientes/id	->  Administrador/Cliente
- DELETE http://locacao/api/v1/clientes/id	->  Administrador


CRUD Equipamento
- GET    http://locacao/api/v1/equipamentos	->  Administrador
- GET    http://locacao/api/v1/equipamentos/id	->  Administrador
- POST   http://locacao/api/v1/equipamentos	->  Administrador
- PUT    http://locacao/api/v1/equipamentos/id	->  Administrador
- DELETE http://locacao/api/v1/equipamentos/id	->  Administrador

Locacao
- GET    http://locacao/api/v1/locacoes		->  Cliente
- GET    http://locacao/api/v1/locacoes/id	->  Cliente
- POST   http://locacao/api/v1/locacoes		->  Cliente
- PUT    *** Não alterar uma locação
